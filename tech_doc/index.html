Introdução

C# é uma linguagem de programação desenvolvida pela Microsoft como parte de sua plataforma .NET.

É uma linguagem moderna, orientada a objetos, e de propósito geral.

Ela é amplamente utilizada para desenvolvimento de aplicativos para Windows e web.

Com uma sintaxe semelhante a outras linguagens C-like, é fácil para desenvolvedores de outras linguagens aprenderem C#.

using System; class Program { static void Main() { } }
  • Orientada a objetos
  • Tipo seguro
  • Multiplataforma
Sintaxe

A sintaxe básica de C# é semelhante a outras linguagens C-like, como C++ e Java.

Os blocos de código são delimitados por chaves.

Os comandos terminam com ponto e vírgula.

int idade = 30; Console.WriteLine("Olá, mundo!");

Tipos de Dados

C# suporta diversos tipos de dados primitivos e complexos.

Os tipos primitivos incluem int, float, double, char e bool.

Tipos complexos incluem arrays e classes.

int numero = 10; string nome = "João";
  • int
  • string
  • bool
Estruturas de Controle

C# possui diversas estruturas de controle de fluxo, como if, for, e while.

Estruturas de controle são usadas para alterar o fluxo de execução do código.

O comando if é usado para executar um bloco de código baseado em uma condição.

if (idade > 18) { Console.WriteLine("Adulto"); } for (int i = 0; i < 10; i++) { Console.WriteLine(i); }
  • if
  • for
  • while
Funções

Funções (ou métodos) em C# são blocos de código que realizam tarefas específicas.

Funções podem ter parâmetros e retornar valores.

É possível definir métodos com diferentes modificadores de acesso.

void ImprimirMensagem() { Console.WriteLine("Mensagem"); } int Somar(int a, int b) { return a + b; }
